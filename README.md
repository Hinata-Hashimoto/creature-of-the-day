# 今日の生き物

今いる場所・今の季節を代表する生き物を、毎日ひとつ教えてくれるWebアプリです。

→ **[公開ページ](https://Hinata-Hashimoto.github.io/creature-of-the-day/)**

---

## 何ができるの？

- 現在地の周辺（半径30km）で、**この時期によく観察される生き物**を1種表示します
- 同じ生き物が続かないよう、過去2週間に表示した種は除外します
- 日付を変えて、季節ごとの生き物の移り変わりを見ることができます

---

## ホーム画面に追加（アプリとして使う）

このサイトはPWA（Progressive Web App）に対応しています。ホーム画面にアイコンを追加すると、ブラウザのUIが消えてアプリのような全画面表示になります。一度読み込んだあとはオフラインでも起動できます。

### iPhone / iPad（Safari）

1. Safari でサイトを開く
2. 画面下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 名前を確認して「追加」をタップ

> Chrome や Firefox では対応していません。**Safari で開いてください。**

### Android（Chrome）

1. Chrome でサイトを開く
2. 右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」または「アプリをインストール」を選択
4. 名前を確認して「追加」をタップ

## データソース

[iNaturalist](https://www.inaturalist.org) の観察データを使用しています。iNaturalistは世界中の市民が投稿する生物観察記録のデータベースで、動植物・菌類・昆虫など幅広い生き物が記録されています。

## 仕組み

### 1. 季節性スコアの計算

iNaturalist API に2種類のクエリを送り、各生き物の「今の時期らしさ」をスコアにします。

```
季節クエリ: 現在地 半径30km × 今週 ±1週（全年分）
年間クエリ: 現在地 半径30km × 年間（全年分）

季節性スコア = 季節観察数² ÷ (年間観察数 + 1)
```

このスコアが高いほど「年間を通じてではなく、この時期に集中して観察される種」になります。スコアが高い順にランキングします。

### 2. 表示する生き物の決め方（直近回避 + 週シャッフル）

ランキングの**上位7種**を今週の候補とします。ただし直近14日間に表示した種は除外します。

```
eligible（候補）= 上位7種 のうち 直近14日間に未表示の種
```

候補をその週のシード値でシャッフルし、曜日（月〜日）に1種ずつ割り当てます。

```
例）第19週の候補: [A, B, C, D, E]
    週シードでシャッフル: [C, A, E, B, D]

    月曜 → C
    火曜 → A  ← 上位種が週のどこかに必ず入る
    水曜 → E
    ...
```

- **同じ日に何度開いても同じ生き物が表示される**（シードが日付固定のため）
- **週が変わると顔ぶれが変わる**（ランキングが更新されるため）
- **2週間のサイクルで繰り返す**（14日回避のため）

### 3. 写真

iNaturalist の各種の代表写真を表示します。クリエイティブ・コモンズライセンス（CC BY・CC BY-NC など）の写真のみ使用し、All Rights Reserved の写真は除外しています。著作権はそれぞれの投稿者に帰属します。

## プライバシー・位置情報について

このアプリは**バックエンドサーバーを持ちません**。取得した位置情報が開発者に送信・記録されることはありません。

位置情報（緯度・経度）は、以下の2つの外部APIへの検索クエリとして使われます。

| 送信先 | 用途 | 通信 |
|---|---|---|
| [iNaturalist API](https://api.inaturalist.org/v1/docs/) | 周辺の生き物を検索 | HTTPS |
| [Nominatim](https://nominatim.org)（OpenStreetMap） | 座標を地名に変換 | HTTPS |

いずれも HTTPS 経由で座標を送るだけで、氏名・アカウント・端末情報などは一切含みません。また、端末内（localStorage）に保存されるのは「どの種をいつ表示したか」という履歴のみで、位置情報は保存されません。

位置情報の取得はブラウザの許可ダイアログで同意した場合のみ行われます。許可しなくても、プリセット都市や手動入力で利用できます。
